# 旅のしおり - 全機能・要件まとめ

## 🎯 アプリ概要

**旅のしおり**は、旅行の計画・実行・記録を一元管理するWebアプリケーションです。複数ユーザーでリアルタイム共有でき、旅行中の予定管理、地図表示、割り勘計算、思い出記録など、旅行に必要な機能を網羅しています。

---

## 📋 主要機能一覧

### 1. **旅行管理**
- **旅行作成ウィザード** — アプリ起動時に旅行名・目的地・期間を入力して新規作成
- **複数旅行管理** — 複数の旅行プランを同時管理可能
- **旅行期間設定** — 日付範囲から自動的にDay1〜DayNを生成
- **Day編集** — Day名の変更、日付の変更、Day順序の並び替え
- **データ永続化** — MySQL DBへの自動保存、起動時は常にDBから読み込み

### 2. **タイムライン（予定管理）**
- **予定追加/編集** — 時間・カテゴリ・タイトル・場所・メモ・予約番号を入力
- **カテゴリ分類** — 移動🚗、食事🍽️、観光🎫、宿泊🏨、その他など
- **ドラッグ&ドロップ並び替え** — タイムライン内で予定を自由に並び替え
- **Google Maps URL自動解析** — URLから住所・座標を自動抽出して「場所」に反映
- **AI場所候補検索** — タイトル入力時にAIが場所候補を提案（「ばくげき」→「直球和食莫逆」）
- **座標特定** — 住所から自動的に緯度経度を取得（Nominatim + Curl フォールバック）
- **添付画像** — 予定に画像を添付してタイムラインに表示（ImageLightbox）
- **ホテル設定** — 各Dayの宿泊先・予約番号を記録
- **移動時間計算** — Google Maps APIで移動時間を自動計算・表示

### 3. **スポットプール**
- **行きたい場所リスト** — 訪問候補地を事前に登録
- **スポット新規作成** — 「＋ スポット追加」ボタンから追加
- **AI場所候補検索** — スポット名からGoogle Maps URLを自動取得
- **スポット管理** — 削除・編集機能

### 4. **地図表示**
- **Leaflet地図コンポーネント** — CartoDB Voyagerタイルを使用
- **全日程表示** — 旅行全体の予定位置をマップに表示
- **当日表示** — 選択したDayの予定のみ表示
- **スポット表示** — スポットプール内の場所をマップに表示
- **ルート線描画** — 日程順に予定をルート線でつなぐ
- **サイドバー開閉** — 地図右側のスポット一覧を開閉可能
- **スポットクリック** — サイドバーのスポットをクリックすると地図がフォーカス（flyTo + ポップアップ）

### 5. **割り勘計算**
- **メンバー管理** — 旅行参加者を追加・削除
- **支出記録** — 支出内容・金額・支払者・対象メンバーを入力
- **支払者必須バリデーション** — 支払者未選択時はエラー表示
- **自動精算計算** — 「AさんがBさんに¥○○払う」という精算結果を自動計算
- **支出一覧表示** — 支出履歴と合計金額を表示

### 6. **AIメモ読み取り**
- **メモインポート** — テキストを貼り付けてAIが自動解析
- **自動振り分け** — AIが日程・場所・カテゴリを判定して各Dayに反映
- **全Day反映** — Day1を選択してメモを読み込んでも、Day1・Day2両方に反映可能
- **精度向上** — 場所名をタイトルに含める、データを適切なフィールドに配置

### 7. **思い出記録**
- **MemoriesView** — 旅行中の写真・動画・テキストを記録
- **タイムスタンプ付き** — 日時を自動記録

### 8. **リアルタイム共有**
- **共有コード発行** — 一意の共有コードを生成
- **共有URL** — 共有URLをコピーして他のユーザーに送信
- **参加機能** — 共有コードを入力して旅行に参加
- **リアルタイム同期** — DBポーリング方式で複数ユーザーの変更を同期
- **共有停止** — 共有を終了してプライベートに戻す
- **接続状態表示** — 接続中のユーザー数を表示

### 9. **PDFエクスポート**
- **旅行記録PDF生成** — タイムライン・スポット・割り勘を含むPDFを生成
- **日本語対応** — jsPDFで日本語フォントをサポート
- **レイアウト最適化** — 見やすいレイアウトで自動ページ分割

### 10. **カウントダウン**
- **旅行開始までの日数** — 旅行開始日までの日数をカウントダウン表示

### 11. **Undo機能**
- **Ctrl+Z対応** — 最後の操作を取り消し

### 12. **URLエンコード共有**
- **URLベース共有** — 旅行データをURLエンコードして共有（DBなしでも動作）

### 13. **ユーザー認証**
- **Manus OAuth** — Manus認証システムで安全にログイン
- **セッション管理** — 自動ログイン・ログアウト機能
- **ユーザープロフィール** — ユーザー情報を表示・管理

### 14. **PWA対応**
- **ホーム画面追加** — スマートフォンのホーム画面に追加可能
- **オフライン対応** — Service Workerで基本機能をオフラインでも利用可能
- **アプリアイコン** — 鮮やかな青い飛行機＋ビビッドピンクの桜の花

### 15. **天気予報**
- **weather.getForecast** — 旅行地の天気予報を取得・表示

## 📊 データモデル

### Trip（旅行）
```
- id: UUID
- name: 旅行名
- destination: 目的地
- days: Day[]
- members: Member[]
- expenses: Expense[]
- spots: Spot[]
- memories: Memory[]
- hotelReservationNo: 宿泊予約番号
```

### Day（日程）
```
- id: UUID
- date: 日付
- name: Day名
- events: Event[]
- hotel: ホテル名
- hotelUrl: ホテルURL
- hotelReservationNo: 宿泊予約番号
- pool: Spot[]
```

### Event（予定）
```
- id: UUID
- time: 時刻
- category: カテゴリ
- title: タイトル
- location: 場所
- lat/lng: 座標
- memo: メモ
- reservationNo: 予約番号
- attachments: 画像URL[]
```

### Member（メンバー）
```
- id: UUID
- name: 名前
```

### Expense（支出）
```
- id: UUID
- title: 内容
- amount: 金額
- payerId: 支払者ID
- coveredMemberIds: 対象メンバーID[]
```

### Spot（スポット）
```
- id: UUID
- name: 場所名
- location: 住所
- lat/lng: 座標
- url: Google Maps URL
```

### Memory（思い出）
```
- id: UUID
- dayId: Day ID
- timestamp: タイムスタンプ
- content: テキスト
- attachments: 画像URL[]
```

### SharedTrip（共有）
```
- id: UUID
- tripId: Trip ID
- code: 共有コード
- createdBy: 作成者ID
- members: ユーザーID[]
```

## 🔄 主要フロー

### 旅行作成フロー
1. アプリ起動
2. ウィザード表示（旅行名・目的地・期間を入力）
3. DBに保存、Day1〜DayNを自動生成
4. タイムライン表示

### 予定追加フロー
1. 「＋ 追加」ボタンをクリック
2. EvtModal表示
3. タイトル入力 → AI場所候補検索
4. Google Maps URLを選択 → 住所・座標を自動抽出
5. その他フィールド（時間・メモ・予約番号・画像）を入力
6. 「保存」ボタン → DBに保存

### 共有フロー
1. 「🔗 共有」ボタンをクリック
2. 共有モーダル表示
3. 「コード発行」ボタン → 共有コード生成
4. 「URLコピー」ボタン → 共有URLをコピー
5. 他のユーザーが共有コードを入力 → 旅行に参加
6. リアルタイム同期開始

### 割り勘フロー
1. 「💴 割り勘」タブをクリック
2. メンバー追加（名前を入力）
3. 支出追加（内容・金額・支払者・対象メンバーを入力）
4. 自動精算計算表示

### メモ読み取りフロー
1. 「📋 メモ読み取り」ボタンをクリック
2. MemoImportModal表示
3. 旅行予定テキストを貼り付け
4. AI解析 → 日程・場所・カテゴリを自動判定
5. 各Dayに予定を自動追加
